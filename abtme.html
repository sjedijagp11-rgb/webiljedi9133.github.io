All About Me!

Selamat datang di website saya! Nama saya adalah Shalom Jedija Gabrielle Purchristantoro, biasa dipanggil Jedi. Saya dari kelas IX-1/33.

Apa tujuan saya membuat web ini?

Proyek ini dibuat sebagai bagian dari tugas akhir untuk mata pelajaran [Sebutkan Mata Pelajaran, cth: TIK/Proyek Ilmiah]. Saya memilih tema SDGs 14: Kehidupan Bawah Laut karena saya percaya bahwa kesehatan laut kita adalah cerminan dari kesehatan planet kita secara keseluruhan.

Saya membuat website ini dengan harapan untuk dijadikan sebuah media edukasi dan informasi untuk meningkatkan kesadaran masyarakat mengenai pentingnya menjaga kelestarian laut sesuai dengan tujuan SDGs 14. Melalui website ini, saya ingin menyampaikan kondisi terkini laut di Indonesia dan dunia, tantangan yang dihadapi, serta kerjasama internasional, peran masyarakat dan pemerintah dalam melindungi ekosistem laut.

Refleksi

Selama mengerjakan tugas ini, saya menyadari bahwa menjaga kelestarian laut bukan hanya tanggung jawab pemerintah atau organisasi-organisasi internasional saja, namun juga tanggung jawab kita sebagai masyarakat yang menggunakan lautan. Melalui proses pembuatan tugas website ini, saya menjadi lebih paham bahwa hal-hal kecil seperti tidak menggunakan kantong plastik saat berbelanja dapat memberi dampak yang cukup besar bagi kehidupan di laut.
